# automatic-code-review
Static Code Analyzer is a Python-based software quality assessment tool that examines source code without executing it. The system identifies syntax errors, coding standard violations, complexity issues, and maintainability concerns to help developers write cleaner and more reliable code.

Features
Syntax Error Detection
Cyclomatic Complexity Analysis
Code Quality Scoring
Naming Convention Validation
Maintainability Checks
Automated Suggestions
Report Generation
Tools and Libraries
Python
Pylint
Radon
AST
JSON
Workflow
